**주제**
현재 사용 중인 학습관리시스템 LMS의 불편사항 개선하여 보다 더 나은 UI/UX를 가진 앱 만들기

**개발 방향**
1. 현재 앱이 가지는 다양한 불편사항 등을 개선
2. 개인 학습뿐만 아니라 팀워크가 필요한 학습을 위한 캘린더 개선
3. 할 일, 알림, 메시지 등 각 페이지들의 개선을 통해 접근성과 직관성 개선

**사용한 기술**
- UI/UX : Android Studio, Kotlin
- 서버 : AWS
